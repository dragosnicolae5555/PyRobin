## PyRobin

This is the python implementation of  https://gitlab.com/raduion/robindialog 

### For a demo please use 
``` python .\main.py precis.mw ```


###  Create conda environment 
``` conda env create -f env.yml ```

###  Activate conda environment 
```activate robin```
